# Jolian Portfolio

A modern personal portfolio website built with React, Vite, and Framer Motion.

## Overview

This portfolio presents my software engineering background, technical skills, selected projects, and contact information through a clean, responsive, and animated user interface.

The goal of this project is to create a professional personal website that reflects my development skills, design attention, and ability to build practical frontend applications.

## Features

- Responsive design for desktop and mobile screens
- Clean and modern user interface
- Animated sections using Framer Motion
- Project showcase section
- Skills and technologies section
- Contact section
- Component-based React structure
- Deployed online using Vercel

## Tech Stack

- React
- Vite
- Framer Motion
- JavaScript
- CSS
- Vercel

## Project Structure

```bash
jolian-portfolio/
├── public/
├── src/
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

## Installation

To run the project locally:

```bash
npm install
npm run dev
```
